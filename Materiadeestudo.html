O que é um Algoritimo

Um algoritmo é uma sequência finita de passos organizada de forma lógica

para resolver um problema. Ele descreve exatamente o que deve ser feito,

em qual ordem e como deve ser feito, para chegar a um resultado correto.

Na programação, antes de escrever o código, é necessário definir

o algoritmo, porque ele serve como base para a solução. O algoritmo

não é a linguagem de programação em si, ele é a ideia estruturada

da solução.

Características

  • Possui inicio e fim.
  • É Lógico e Organizado.
  • Criado para resolver uma situaçao determinada

Exemplos

Exemplo-1

Somar dois números: ler dois valores, calcular a soma e mostrar o resultado.

Exemplo-2

Calcular média: ler duas notas, somar, dividir por dois e exibir a média.

Exemplo-3

Verificar número par: ler um número, dividir por dois e informar se o resto é zero ou não.
